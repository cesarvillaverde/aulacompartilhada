##         ---------------------------------**BEBE MUITO MAIS APP**-------------------------
###         -----Aplicativo simples para listagem e visualização de bebidas, permitindo ao usuário explorar diferentes opções, ver detalhes e filtrar por categorias.----
###    -------O BEBE MUITO MAIS App é um projeto desenvolvido com um fim comercial, com o objetivo de aumentar o consumo de bebidas.
### Funcionalidades
1. Listagem de bebidas
2. Visualização de detalhes da bebida
3. Filtro por categoria
4. Interface amigável
### Tecnologias Utilizadas
- Linguagem: Python
- Framework: React / React Native / Flutter
### Melhorias futuras
* Sistema de favoritos
* Busca por nome da bebida
* Integração com API externa
